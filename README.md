# ClaudeClaw Пантеон

> Твой первый AI-агент в Telegram. Программист-перфекционист на Claude Code.

---

## Что это

AI-агент **Тваштар** - программист с характером, работающий через Telegram. Построен на [ClaudeClaw](https://github.com/earlyaidopters/claudeclaw) - обёртке вокруг Claude Code CLI.

**Не чат-бот.** Тваштар запускает настоящий `claude` CLI на твоём Mac/Linux и отправляет результат в Telegram. Все твои файлы, скиллы и инструменты - доступны с телефона.

**Начинаем с одного - растём до команды.** Этот репозиторий разворачивает одного агента-программиста. Когда освоишься - добавишь маркетолога, исполнителя, координатора. Инфраструктура готова.

## Установка за 5 минут

### 1. Создай Telegram-бота

Открой [@BotFather](https://t.me/botfather) в Telegram:
1. Напиши `/newbot`
2. Дай имя (например: "Тваштар")
3. Дай username (например: `my_tvashtar_bot`)
4. Сохрани токен - он понадобится через минуту

### 2. Узнай свой Telegram ID

Напиши [@userinfobot](https://t.me/userinfobot) в Telegram - он ответит числом. Сохрани.

### 3. Клонируй и установи

```bash
git clone https://github.com/YOUR_USERNAME/claudeclaw-pantheon.git
cd claudeclaw-pantheon
npm install
```

### 4. Настрой ключи

```bash
cp .env.example .env
```

Открой `.env` и заполни два обязательных поля:
```
TVASHTAR_BOT_TOKEN=токен_от_BotFather
ALLOWED_CHAT_ID=твой_telegram_id
```

### 5. Залогинься в Claude

```bash
claude login
```

### 6. Собери и запусти

```bash
npm run build
npm run agent:start tvashtar
```

### 7. Проверь

Напиши своему боту в Telegram. Если отвечает - всё работает.

## Как добавить второго агента

Когда будешь готов расти - добавляй новых агентов:

```bash
npm run agent:create
```

Или вручную: скопируй `agents/_template/`, заполни `agent.yaml` и `CLAUDE.md`.

### Рекомендуемая команда (Пантеон)

| Агент | Роль | Модель | Зачем |
|-------|------|--------|-------|
| **Индра** | Координатор | Opus | Главный, раздаёт задачи |
| **Манас** | Исполнитель | Haiku | Быстрые рутинные задачи (дёшево) |
| **Кама** | Маркетолог | Sonnet | Контент, тренды, копирайтинг |
| **Тваштар** | Программист | Opus | Код, архитектура, тесты |
| **Дхарма** | ОТК / Юрист | Sonnet | Проверка качества, документы |

Каждый новый агент - это:
1. Папка в `agents/`
2. Файл `agent.yaml` (модель, токен, скиллы)
3. Файл `CLAUDE.md` (характер, роль, инструкции)
4. Токен бота от @BotFather в `.env`
5. `npm run agent:start <id>` для запуска

## Память и Hive Mind

Все агенты делят **одну SQLite базу** (`store/claudeclaw.db`). Даже с одним агентом - память работает.

### Как устроена память

**3 слоя поиска:**
1. **Vector search** - семантический поиск по смыслу (нужен OpenAI API key)
2. **FTS5 keyword** - полнотекстовый поиск (работает без ключей)
3. **High-importance** - недавние важные воспоминания

**Hive Mind** - общая доска действий:
- Агент записывает что сделал (agent_id, action, summary)
- Когда добавишь других агентов - они увидят историю

### OpenAI API ключ (для embeddings)

Для полноценной векторной памяти добавь `OPENAI_API_KEY` в `.env`. Без него память работает через keyword search - проще, но менее точно.

## Персонализация Тваштара

### Характер
Редактируй `agents/tvashtar/CLAUDE.md` - там характер, стиль общения, фразы. Сделай его своим.

### Скиллы
Редактируй `agents/tvashtar/agent.yaml` - добавляй или убирай скиллы.

### Модель
В `agent.yaml` поле `model`. Варианты:
- `claude-opus-4-6` - самая мощная (для кода лучший выбор)
- `claude-sonnet-4-20250514` - быстрее и дешевле
- `claude-haiku-4-5` - самая быстрая и дешёвая

## FAQ

**Нужен ли API ключ Anthropic?**
Нет. Если у тебя Claude Max - используется твоя подписка через `claude login`.

**Можно ли запустить без главного бота?**
Да. `npm run agent:start tvashtar` запускает только Тваштара. Главный бот не нужен.

**Что если я на Free плане Claude?**
Работать будет, но код на Opus получается значительно лучше. Рекомендуем Max.

**Работает на Windows?**
Через WSL2 - да.

**Как остановить?**
`Ctrl+C` в терминале. Или `npm run status` для проверки.
